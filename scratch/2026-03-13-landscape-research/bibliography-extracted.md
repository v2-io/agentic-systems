# Extracted Bibliography from Landscape Research

**Date**: 2026-03-13
**Source**: Research agents during comparative theory analysis session
**Purpose**: Raw reference collection for integration into ACT's prior art positioning, reference catalog, and potential collaboration targets.

**Caveat on the novelty analysis**: The `novelty-analysis-raw.md` file contains results from agents that were given paraphrased ACT claims stripped of context and epistemic labels. ACT explicitly identifies most of these (e.g., η* as Kalman gain, timescale separation as Tikhonov's theorem) as established results being synthesized, NOT as novel discoveries. The novelty analysis answered "is this formula new?" when the actual question should have been "is this specific cross-domain integration new?" See the main discussion transcript for the full methodological critique.
