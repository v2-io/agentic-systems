### A.1 Factor out the sector-Lyapunov template as a shared lemma

**Problem:** Every persistence-flavored result in AAD — single-agent epistemic (`#persistence-condition`), strategic (`#strategy-persistence-schema`), team (`#team-persistence`), composite closure (`#composition-closure` bridge lemma), coordination overhead (`#tempo-composition`), adversarial destabilization (`#adversarial-destabilization`) — has the form "correction rate > effective disturbance rate / tolerance," proved via a Lyapunov function and sector-bounded correction. The theory currently rebuilds this pattern five times with slightly different notation and caveats. Per Opus audit "Bigger-picture synthesis §1," this obscures that *the theory is one result about bounded-correction dynamics applied to five different state spaces.*

**Repair:** Create a new Appendix segment `#sector-persistence-template` that states the single Lyapunov argument once, in parameter-free form:

> *[Template]* Given a state variable $\xi$ with dynamics driven by a bounded correction function $F$ satisfying the sector condition $\xi^T F(\xi) \geq \alpha \lVert\xi\rVert^2$ within region $\lVert\xi\rVert \leq R$, and a disturbance rate $\rho_\xi$, structural persistence requires $\alpha \gt \rho_\xi / R$. The ultimately bounded radius is $R^\ast = \rho_\xi / \alpha$ (Model D) or $R^\ast_S = \sigma_\xi\sqrt{n/(2\alpha)}$ (Model S).

Then convert the existing persistence-flavored segments to two-paragraph instantiations citing the template:

- `#persistence-condition` — $\xi = \delta_t$, $F = $ correction function, $\rho_\xi = \rho$
- `#strategy-persistence-schema` — $\xi = \delta_\Sigma$, $F_\Sigma$, $\rho_\Sigma$; cite template for structural parallel (but see A.2 — add forgetting prerequisite)
- `#team-persistence` — $\xi = \delta_i$, with effective disturbance decomposition $\rho_i^{\text{eff}} = \rho_{i,\text{env}} + \sum_j \gamma^{\text{adv}}_{j\to i}\mathcal T_j - \sum_j \gamma^{\text{coop}}_{j\to i}\mathcal T_j$
- `#composition-closure` bridge lemma — $\xi = e_t$ (trajectory error), $\rho_\xi = \varepsilon^\ast \nu_c$, bound $L\varepsilon^\ast/\alpha_c$
- `#tempo-composition` — $\xi = \delta_c$ with effective disturbance $\rho_{\text{eff}} = \rho_{\text{ext}} + \varepsilon^\ast\nu_c$; tempo penalty $\varepsilon^\ast\nu_c / \lVert\delta_{\text{critical}}\rVert$
- `#adversarial-destabilization` — Model D: $\rho_B = \rho_{B,\text{base}} + \gamma_A\mathcal T_A$; Model S: $\sigma_B = \sigma_{B,\text{base}} + \gamma_A\mathcal T_A$. Both instantiations with the same template.

**Subsumption note:** This move absorbs Opus "Bigger-picture synthesis §6" (persistence and destabilization as single statement — the structural persistence boundary is just the template with effective disturbance). It also sharpens the Codex Finding 2 scope automatically: "applies at every level" becomes "applies at every level where the template's preconditions hold — namely, (A4) sector bound, the tier-specific contraction where relevant, and now the scope condition."

**Estimated effort:** One full session. The Lyapunov content already exists in `#sector-condition-derivation` (Prop A.1 and A.1S) — this work is refactoring, not new derivation.

**Lint watch:** The template segment will need bar-notation discipline from the start — use `\lvert`/`\rvert` and `\lt`/`\gt` throughout.

### A.2 Opus Finding 2 — forgetting as prerequisite in `#strategy-persistence-schema`

**Problem:** The schema claims a structural identity between epistemic and strategic persistence, but the strategic case has $\alpha_\Sigma$ intrinsically decaying with experience ($1/(n+1)$ scaling from Props B.1–B.6). For any fixed $(\rho_\Sigma, R_\Sigma)$, every agent eventually crosses below threshold *unless it discounts*. The spike `msc/spike-single-edge-strategic-dynamics.md` (§5, §9) states this as a **promotion-blocking prerequisite**; the segment has it as a one-paragraph Working Note via `strategic-dynamics-derivation.md` only.

**Repair:** Add to `#strategy-persistence-schema` Formal Expression:

> *[Formulation (forgetting prerequisite)]* The form $\alpha_\Sigma \gt \rho_\Sigma / R_\Sigma$ is an **instantaneous** persistence check at the current experience level. Without a forgetting mechanism, $\alpha_\Sigma$ decays monotonically with experience ($\propto 1/(n+1)$ for Beta-Bernoulli) and every agent eventually violates the threshold. Exponential forgetting with rate $\lambda$ stabilizes $\alpha_\Sigma$ at approximately $1 - \lambda$, converting the instantaneous check to a trajectory guarantee with the constraint:
>
> $$(1 - \lambda) \gt \rho_\Sigma / R_\Sigma$$
>
> This is a prerequisite of the schema, not a heuristic. An agent without forgetting has no long-run strategic persistence guarantee regardless of its initial $\alpha_\Sigma$.

Update Epistemic Status to reflect that the structural identity with `#persistence-condition` holds *under forgetting*.

**Estimated effort:** 30 minutes. Content already derived in `msc/spike-single-edge-strategic-dynamics.md` §5 and §9.
